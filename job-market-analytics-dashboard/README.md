# Job Market Analytics Dashboard

A portfolio-ready data analytics dashboard that ingests live job posting data from a public jobs API and visualizes labor-market trends.

## Features

- Live API ingestion
- Interactive dashboard
- Company analytics
- Location analytics
- Remote-job analysis
- Data exploration table

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- REST APIs

## API

This project currently uses:

- Arbeitnow Job Board API

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
