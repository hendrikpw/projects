# Data Science Portfolio Projects

This repository contains portfolio-ready data science and analytics projects. Each project is structured as a small product with API ingestion, data processing, analysis, visualization, and documentation.

## Projects

| Project | Description | Stack |
|---|---|---|
| [Job Market Analytics Dashboard](./job-market-analytics-dashboard) | Streamlit dashboard that collects job postings from the Arbeitnow API and analyzes labor-market demand by keyword, location, company, remote status, and skills. | Python, Streamlit, Pandas, Plotly, API ingestion |

## Repository structure

projects/

## Goal

The goal of this repository is to demonstrate practical data science skills through real-world API projects, including data ingestion, transformation, exploratory analysis, dashboarding, and later machine learning extensions.
